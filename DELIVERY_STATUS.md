# 전달 상태

## 현재 기준

- 정식 프런트엔드: `apps/frontend`
- 백엔드: `apps/api`
- 강사용 대시보드, 학생 목록, 학생 상세, 시험 관리, 목표 대학 정책, 학생용 대시보드 연결 완료

## 확인 완료 항목

- 백엔드 health 응답 확인
- 프런트엔드 프로덕션 빌드 확인
- `/frontend/*` 화면 데이터 경로 정리
- `/auth/*` 인증 경로 정리
- 학생 결과 저장과 전략 재계산 연결
- 대학 정책 등록, 수정 연결

## 이번 단계 보강 항목

- 서버 시작 시 자동 `create_all` 의존 제거
- 명시적 데이터베이스 초기화 명령 추가
- 로컬 SQLite와 운영 PostgreSQL 기준 분리
- 실행 문서와 환경 변수 갱신

## 남은 마감 항목

- 마이그레이션 도구 도입 준비
- 테스트 범위 확대
- 남은 사용자 노출 문구 최종 점검
