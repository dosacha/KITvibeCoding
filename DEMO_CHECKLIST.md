# 데모 체크리스트

## 실행 준비

- 백엔드 실행 확인
- 프런트엔드 실행 확인
- 테스트 계정 확인
- `python manage_db.py seed` 실행 여부 확인

## 강사 시연 흐름

1. 강사 계정 로그인
2. 강사용 요약 화면 확인
3. 학생 목록 진입
4. 학생 상세 진입
5. 학생 결과 입력
6. 전략 다시 계산
7. 시험 관리에서 시험과 문항 등록
8. 목표 대학 정책 등록 또는 수정

## 학생 시연 흐름

1. 학생 계정 로그인
2. 나의 학습 전략 확인
3. 진단 근거 확인
4. 우선 보완 과목 확인
5. 우선 보완 단원 확인

## 확인 사항

- 모든 사용자 노출 문구가 쉬운 한국어인지 확인
- 로그인 실패 때 한국어 오류 문구가 보이는지 확인
- 학생 상세에서 문항별 정오답 입력이 가능한지 확인
- 학생 결과 저장 뒤 전략이 다시 계산되는지 확인
- 시험 관리에서 시험과 문항 등록이 가능한지 확인
- 대학 정책 등록과 수정이 가능한지 확인
- 조용한 mock 전환 없이 실제 오류가 표시되는지 확인
